작성일: 2026-03-11  
문서 상태: Draft

---

# 1. 제품 개요

본 시스템은 **한국투자증권 계좌를 연동하여 국내 주식 자동매매를 수행하는 개인용 프로그램**이다.

시스템은 다음 기능을 제공해야 한다.

- 종목 리스트 자동 생성
- 종목 스캔 기반 전략 실행
- 점수 기반 종목 선택
- 리스크 관리 기반 자동 주문
- 매매 기록 저장
- 장애 대응(Fail-Safe)
- 텔레그램 알림

---

# 2. 시스템 목표

1. 사용자가 설정한 전략을 기반으로 **완전 자동매매 수행**
2. **리스크 통제 우선 구조**
3. 전략 실행과 주문 실행을 **명확히 분리**
4. 장애 발생 시 **주문 사고 방지**
5. 매매 발생 시 **실시간 알림 제공**

# 6. 종목 리스트 관리

종목 리스트는 **시장 지표 기반으로 자동 생성**한다.

## 대상 시장

- 코스피
- 국내 ETF

## 종목 필터 조건

| 조건 |
|---|
| 주가 ≥ 3,000원 |
| 20일 평균 거래대금 ≥ 50억 |
| 거래대금 상위 150 종목 |

## 업데이트 시점

- 매 거래일 장 시작 전 (08:50)

# 7. 시장 데이터 처리

| 항목 | 내용 |
|---|---|

| 데이터 저장 | 로컬 캐시 저장 |
| 전략 스캔 주기 | 30초 |

# 8. 종목 선택 알고리즘

## 점수 기반 평가

각 종목은 전략 지표를 기반으로 점수를 계산한다.

### 사용 지표

| 지표 | 설명 |
|---|---|
| 거래량 증가 | 수급 유입 확인 |
| 가격 모멘텀 | 최근 상승률 |
| 20일 이동평균 | 추세 판단 |
| MA5 > MA20 | 상승 추세 확인 |
| ATR 변동성 | 변동성 측정 |
| RSI | 과매수 / 과매도 판단 |

총점: **100점**

---

## 매수 조건

| 조건 |
|---|
| 점수 ≥ 70 |
| 가격 > MA20 |
| MA5 > MA20 |

## 매수 절차

1. 종목 스캔
2. 점수 계산
3. 점수 70 이상 종목 필터
4. 점수 기준 정렬
5. 상위 종목 매수

# 15. 텔레그램 알림

시스템은 매매 및 시스템 이벤트 발생 시 텔레그램 알림을 전송해야 한다.

## 매수 체결 알림

- 종목명
- 종목코드
- 전략명
- 체결 가격
- 체결 수량
- 체결 시각
- 보유 종목 수

## 매도 체결 알림

- 종목명
- 체결 가격
- 손익 금액
- 손익률

## 시스템 알림

| 이벤트 |
|---|
| 손절 발생 |
| 익절 발생 |
| 자동매매 시작 |
| 자동매매 중지 |
| 일일 손실 한도 도달 |
| 장애 발생 |
| 장애 복구 |

## 알림 규칙

- 체결 확인 시점에 전송
- 동일 체결 중복 알림 금지
- 알림 실패 시 로그 기록
- 알림 실패가 매매 로직에 영향을 주지 않아야 함
